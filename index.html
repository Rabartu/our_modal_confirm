Description

This plugin display a modal dialog used for confirm operations like delete, add, or edit among others...

  • Easy to use.
  • Three types of hooks ( url, form and callback).
  • Setup in one line.
  • High customization.
  • Customization of Options.
  • Customization of Styles.

Installation

Installing this plugin is super-easy, just copy the file modalconf.js and include it in your page. 

<script src="assets/js/modalconf.js"></script>

Requirements

  • This plugin require the jquery library.

Usage

Via Attributes

Activate the modal without writing JavaScript.

Set class="modal-confirm" on a controller element, like a button, add a data-title="title" to set header title and data-content="content body" to set the content. Also, you can add the attribute data-options={opts} to overwrite the default options.

Example

<a class="modal-confirm"
            data-title="Search Confirmation" data-content="Search in Google ?" data-options='{"confirmType" : "url" , "confirmArgs" : "http://www.google.com" }'
            >Modal Confirm</a>
Launch Example

Via JavaScript

Call a modal with a single line of JavaScript:

Example

$('#modalLauncher').modalconf('show',{ titleText: "Search Confirmation" ,contentText: "Search in Google ?"
            ,confirmType: "url" ,confirmArgs:
            "http://www.google.com"});
Launch example

Options

Name type default description
fxSpeed int 300 Duration of in animation.
hideOnClick bool true Define if the modal can close when user click in external area.
confirmType string url Type of confirmation:
  • url
  • form
  • callback
confirmArgs string NULL Arguments of confirmType:
  • if confirmType is url: confirmArgs represent the url, ex: http://www.google.com.
  • if confirmType is form: confirmArgs represent the selector for make form submit, ex: #form-example.
  • if confirmType is callback: confirmArgs represent a functions and arguments separated by coma, ex: confirm;You agree ! 'confirm' is the function and 'You agree !' the 1st argument.
titleText string NULL Modal 's title.
contentText string NULL Modal 's text container.
closeText string x Text in the close button.
confirmText string Confirm Text displayed in the confirm button.
cancelText string Cancel Text displayed in the cancel button.

Customize

The plugin have some options to customize the appareance:

Name type default description
overlayOpacity float 0.8 Opacity of the windows when show the modal, the value is betwen 0.0 ( opacity null) y 1.0 (transparency)
width string 60% Width in percent
height string 40% Width in percent
borderRadius string 6px Radius Border of the Modal
background string #FFFFFF Background Color
color string #666 Text Color
btnCloseClass string NULL Add custom class to close button
btnConfirmClass string NULL Add custom class to confirm button
btnCancelClass string NULL Add custom class to cancel button
